# Pixel-Genie 🧞‍♂️

**AIが魔法のように生み出す究極のピクセルアートスタジオ**

テキストや画像から高品質なピクセルアートを瞬時に生成・編集できる、クリエイター向けツールです。

![Pixel-Genie デモ](https://github.com/figyaro/Pixel-Genie/blob/main/public/screenshot.png?raw=true)

## ✨ 主な機能

- **テキスト → ピクセルアート生成**  
  Gemini AIで「レトロな剣士」「かわいい猫のドット絵」など、プロンプトから即生成
- **画像 → ピクセルアート変換**  
  写真やイラストをアップロードして美しいピクセルスタイルに自動変換
- **本格ピクセルエディタ**  
  レイヤー機能、Undo/Redo、塗りつぶし、消しゴム、選択ツールなど充実
- **自由な解像度**  
  16×16 〜 128×128 まで対応
- **パレット最適化 & PNGダウンロード**
